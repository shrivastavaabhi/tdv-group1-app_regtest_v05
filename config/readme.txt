Below is the format of the set command to be used in the set files.

-set <DS_PATH> <TYPE> <VARIABLE_TO_SET> <VARIABLE_VALUE>
-set <DS_PATH> <TYPE> <VARIABLE_TO_SET> <VARIABLE_VALUE>
-set <DS_PATH> <TYPE> <VARIABLE_TO_SET> <VARIABLE_VALUE>
-set <DS_PATH> <TYPE> <VARIABLE_TO_SET> <VARIABLE_VALUE>
-set <DS_PATH> <TYPE> <VARIABLE_TO_SET> <VARIABLE_VALUE>

Here is the example

-set /shared/TEP/WaybillEntity_v02/1.PhysicalLayer/DS-DB2-OPTCS-IntermodalTrucker DATA_SOURCE host HOSTNAME
-set /shared/TEP/WaybillEntity_v02/1.PhysicalLayer/DS-DB2-OPTCS-IntermodalTrucker DATA_SOURCE port PORT
-set /shared/TEP/WaybillEntity_v02/1.PhysicalLayer/DS-DB2-OPTCS-IntermodalTrucker DATA_SOURCE database DATABASE_NAME
-set /shared/TEP/WaybillEntity_v02/1.PhysicalLayer/DS-DB2-OPTCS-IntermodalTrucker DATA_SOURCE user USERID
-set /shared/TEP/WaybillEntity_v02/1.PhysicalLayer/DS-DB2-OPTCS-IntermodalTrucker DATA_SOURCE password ENCRYPTED_PASSWORD
